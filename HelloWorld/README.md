# Hello World

Este programa imprime el texto *Hello world!*

<br>

Para analizar el codigo se utiliza el comando ghdl con la opción -a
```sh
ghdl -a hello.vhdl
```
Ahora se elabora la unidad con la opción -e
```sh
ghdl -e hello
```
Y se ejecuta la unidad con la opcion -r
```sh
ghdl -r hello
```
